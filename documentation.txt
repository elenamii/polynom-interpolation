#poly_add theorie
# p= [1,2,3] --> 1*x^0 + 2*x^1 + 3*x^2 
# q = [5,6] --> 5*x^0 + 6*x^1 

# p(x) + q(x) = (1+5)*x^0 + (2+6)*x^1 + (3+0)*x^2 
#             = (1+5) + (2+6)x + (3+0)x^2 
#              --> array aus koeffizienten: [6, 8, 3]

#poly_add Funktion
INPUT: 
    - p: Liste von Koeff. eines Polynoms
    - q: Liste von Koeff. eines anderen Polynoms
OUTPUT: 
    - Liste von Koeff. des Summenpolynoms
BESONDERHEITEN: 
    - Wenn ein Polynom kürzer ist (also kleiner als max_lenght), werden die fehlenden Koeff. wie Nullen behandelt 
    - try/except IndexError wird verwendet um zu erkennen, dass ein Polynom kürzer ist. wenn man auf einen nicht vorhandenen Index zugreift, kommt der Fehler „Index out of range“, und in diesem Fall wird der fehlende Koeff. automatisch auf 0 gesetzt
